# Agoric Airdrop Factory

## Getting started 

```bash
# clone repo
$ git clone https://github.com/lari-labs/dapp-meme-coin && cd dapp-meme-coin
# install deps
$ dapp-meme-coin ## agoric install
```

### Running tests

```bash
# cd into contract
cd contract
# run all tests
$ contract ## yarn run test
# run specific test
$ npx ava ./test/<specific-test-file>.js 
```
